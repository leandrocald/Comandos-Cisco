configuração do roteador 

no ip domain-loopkup (atribuir ao roteador para não pesquisar nome de host quando digitar algo errado)


ena
conf t
hostname R1
no ip domain-loopkup
security passwords min-length 10 (atribuir nº de tentativas)
enable secret cisco12345
line con 0
password ciscoconpass
exc-timeout 5 0 (tempo de tentativa)
login
logging syncronous (sincroniza o debug e a saída do software cisco ios e impede que essas 
mensagens interropam no teclado)
exit
